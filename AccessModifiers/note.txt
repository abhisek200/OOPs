=>=> There are only 3:

1. public
Anyone can access it
Think: open to all.

2. private
Only the class itself can access it
Think: personal diary.

3. protected
Data and methods are accessible inside class and to its derived class
Class + its child classes can access it

public = everyone
private = only me
protected = me + my children
